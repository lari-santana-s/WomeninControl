# 💙 WomeninControl

Sistema de controle financeiro desenvolvido em Python com foco na organização financeira de mulheres, promovendo autonomia e consciência sobre gastos.

---

## 📌 Sobre o projeto

O WomeninControl é uma aplicação em terminal que permite:

- Cadastro de usuário
- Registro de entradas financeiras
- Organização de gastos por categorias
- Identificação de possíveis economias
- Visualização do saldo mensal

---

## 🚀 Funcionalidades

- Cadastro de usuário
- Validação de senha
- Registro de renda
- Registro de gastos por categorias
- Categorias pré-definidas
- Cálculo automático de saldo
- Listagem de gastos (maior → menor)
- Identificação de possíveis economias

## ▶️ Como executar

1. Clone o repositório:

git clone https://github.com/lari-santana-s/WomeninControl.git

2. Acesse a pasta:

cd WomeninControl

3. Ative o ambiente virtual (opcional):

venv\Scripts\activate

4. Execute o projeto:

python app/main.py
